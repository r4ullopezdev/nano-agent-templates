# nano-agent-templates

[![CI](https://github.com/r4ullopezdev/nano-agent-templates/actions/workflows/ci.yml/badge.svg)](https://github.com/r4ullopezdev/nano-agent-templates/actions/workflows/ci.yml)
[![License](https://img.shields.io/badge/license-Apache--2.0-blue.svg)](./LICENSE)

Reusable workflow templates and prompt contracts for multi-agent organizational patterns.

## Included templates

- content ops department
- research department
- customer support triage
- marketing workflow
- internal operations assistant

Each template includes:

- config
- roles
- prompt contracts
- execution flow
- expected outputs
- sample run

## Quickstart

```bash
npm install
npm run test
```

Use the template folders directly as starting points for custom workflows.
